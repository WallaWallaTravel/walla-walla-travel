## How to Use

### **Uploading Media**
1. Click "+ Upload Media" button
2. Select a file (JPG, PNG, WebP, GIF, MP4, WebM)
3. Choose a category (Winery, Service, Vehicle, Location, Brand)
4. Add subcategory (optional, e.g., "lecole-no-41")
5. Enter title (required)
6. Add description (optional, helps with organization)
7. Enter alt text (required, for accessibility and SEO)
8. Add tags (optional, comma-separated, e.g., "wine, tasting, outdoor")
9. Check "Set as hero image" if this should be featured
10. Click "Upload Media"

### **Editing Media**
1. Find the media using filters/search
2. Click the "Edit" button on the card
3. Update any details:
   - Category
   - Subcategory
   - Title
   - Description
   - Alt text
   - Tags
   - Hero status
4. Click "Save Changes"

### **Deleting Media**
1. Click the üóëÔ∏è button on any media card
2. Confirm deletion
3. Media is permanently removed

### **Finding Media**
1. **By Category:** Use the category dropdown
2. **By Search:** Type in the search box (searches title, filename, tags)
3. **By Tag:** Search for a specific tag
4. **Combine Filters:** Use category + search together

## Best Practices

### **Organizing Media**

1. **Use Clear Titles**
   - ‚úÖ Good: "L'Ecole No 41 Tasting Room"
   - ‚ùå Bad: "IMG_1234"

2. **Add Subcategories**
   - For wineries: Use winery slug (e.g., "lecole-no-41")
   - For services: Use service type (e.g., "wine-tours")
   - For vehicles: Use vehicle model (e.g., "sprinter-van")

3. **Write Descriptive Alt Text**
   - ‚úÖ Good: "Historic schoolhouse winery tasting room with red brick exterior"
   - ‚ùå Bad: "Winery photo"

4. **Tag Consistently**
   - Use lowercase
   - Be specific: "wine", "tasting", "outdoor", "summer", "vineyard"
   - Think about how you'll search later

5. **Set Hero Images**
   - Mark your best photo for each category/subcategory
   - Hero images are featured prominently in proposals

### **File Guidelines**

- **Images:** At least 1920px wide for best quality
- **File Size:** Keep under 5MB for fast loading
- **Format:** JPG for photos, PNG for graphics, WebP for web optimization
- **Videos:** Keep under 50MB, use MP4 format

### **SEO & Accessibility**

- Always fill in alt text (helps with SEO and screen readers)
- Use descriptive titles
- Add relevant tags
- Include location names when applicable

## Common Workflows

### **Preparing for a Proposal**
1. Search for relevant wineries: Search "winery-name"
2. Filter by category: "Winery"
3. Select hero images for each winery
4. Check that all have good alt text

### **Organizing New Photos**
1. Upload all photos at once
2. Use consistent subcategories
3. Batch-tag related photos
4. Set hero images for key locations

### **Spring Cleaning**
1. Switch to List View
2. Sort by view count
3. Delete unused media
4. Update outdated photos

## Tips & Tricks

- **Auto-Select on Upload:** When you select a file, the title and alt text auto-populate with the filename
- **Quick Search:** Start typing in the search box for instant results
- **View Count:** Shows how popular each image is (useful for identifying best performers)
- **Grid vs List:** Use Grid for visual browsing, List for detailed management
- **Tags for Everything:** The more tags, the easier to find later

## Troubleshooting

### **Can't find a photo**
- Check your category filter (set to "All Media")
- Clear the search box
- Check spelling in search

### **Upload failed**
- Check file size (must be under 50MB)
- Verify file type (JPG, PNG, WebP, GIF, MP4, WebM only)
- Ensure you filled in all required fields (title, alt text)

### **Edit button gives 404**
- This has been fixed! Refresh the page and try again
