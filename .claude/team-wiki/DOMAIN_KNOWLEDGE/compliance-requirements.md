# Compliance Requirements

## Motor Carrier Context

**NW Touring & Concierge** (Northwest Touring LLC) is a regulated motor carrier:
- **USDOT**: 3603851
- **MC**: 1225087
- **Operation**: Charter & Tour, Passenger
- **Scope**: Interstate AND Intrastate

## FMCSA Requirements

### Driver Qualification File (DQ File)

Every driver must have a complete DQ file containing:

| Document | Requirement | Renewal |
|----------|-------------|---------|
| Application for Employment | Initial hire | Keep 3 years after termination |
| MVR (Motor Vehicle Record) | Annual review | Every 12 months |
| Medical Certificate | DOT physical | Every 24 months (or as specified) |
| CDL Copy | Valid for vehicle class | Before driving, verify annually |
| Road Test Certificate | Or equivalent experience | One-time |
| Drug & Alcohol Testing | Pre-employment + random | Ongoing program |

### Hours of Service (HOS)

**For passenger-carrying vehicles:**
- **10-hour driving limit** after 8 consecutive hours off duty
- **15-hour on-duty limit** after 8 consecutive hours off duty
- **60/70-hour limit** in 7/8 consecutive days

### Vehicle Inspections

**Pre-Trip Inspection (DVIR - Driver Vehicle Inspection Report):**
- Required before each trip
- Must cover all safety-critical items
- Document defects found
- Certify vehicle is safe to operate

**Post-Trip Inspection:**
- Required at end of each day's work
- Document any defects discovered during operation
- Report issues for repair before next use

### Safety Items to Inspect

| Category | Items |
|----------|-------|
| Brakes | Service brakes, parking brake, connections |
| Lights | Headlights, taillights, turn signals, markers |
| Tires | Condition, pressure, lug nuts |
| Mirrors | Condition, adjustment |
| Emergency | Fire extinguisher, triangles, first aid |
| Steering | Free play, alignment |
| Fuel System | Leaks, cap, tank |
| Windshield | Wipers, washer, cracks |

## Compliance Status Tracking

**Auditor's Dream tracks:**
- Requirement status (compliant/expiring/non-compliant)
- Document expiration dates
- Automatic reminders for renewals
- Aggregated compliance scores

## Integration with Walla Walla Travel

When a tour is booked:
1. System can verify driver is qualified
2. System can verify vehicle is inspected
3. DVIRs linked to booking for audit trail
4. Hours tracked to prevent HOS violations
